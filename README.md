# Automated Cooking Method Recognition for Indian Food Images
## Deep Learning Research Project

### Overview
This repository contains the implementation of our research paper:
"Automated Cooking Method Recognition for Accurate Calorie Estimation in Indian Food Images Using Deep Learning Texture Analysis"

We propose an EfficientNet-B0 based system that automatically detects whether Indian food was fried, steamed, or grilled from a single photo. This solves a key limitation in existing dietary tracking systems which require users to manually input cooking method.

### Research Contribution
- First study specifically addressing automated cooking method detection for Indian cuisine
- Purpose-built dataset of 9,672 balanced images across 3 cooking method classes
- Two-stage EfficientNet-B0 fine-tuning approach outperforms ResNet-50 baseline
- Direct improvement to calorie estimation accuracy
- Result metrics and plots are **included in this repository** (`results/`) for full reproducibility — no need to re-run training to inspect findings

### How to Reproduce

#### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

#### Step 2 — Download datasets
Download these two datasets:
1. Indian Food Images: [kaggle.com/datasets/iamsouravbanerjee/indian-food-images-dataset](https://www.kaggle.com/datasets/iamsouravbanerjee/indian-food-images-dataset)
2. Food-101: [kaggle.com/datasets/dansbecker/food-101](https://www.kaggle.com/datasets/dansbecker/food-101)

Extract to your Downloads folder maintaining the original structure.

#### Step 3 — Build dataset
```bash
python scripts/data_preparation/build_final_dataset.py
python scripts/data_preparation/prepare_dataset.py
python scripts/data_preparation/check_dataset.py
python scripts/data_preparation/augment_dataset.py
```

#### Step 4 — Train models
> Recommended: run on GPU (Google Colab or similar)

```bash
python scripts/training/train_baseline.py
python scripts/training/train_proposed_model.py
```

#### Step 5 — Results
Results are saved to `results/metrics/` and `results/plots/`

## Experiments

### Phase 4 — Calorie Estimation Validation
Run the calorie experiment to prove cooking method
detection improves calorie estimation accuracy:

```bash
python scripts/experiments/calorie_experiment.py
```

Results saved to:
- results/calorie_experiment.json
